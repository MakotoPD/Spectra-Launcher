## Changes (v0.4.1)

### 🐛 Bug Fixes
- **Window controls**: Added window control to MacOS and Linux 

## Changes (v0.4.0)

### ✨ Features & Improvements
- **Smart dependency cleanup**: when you delete a mod, Spectra now checks for dependencies it pulled in and offers to remove the ones nothing else needs. A dialog lists them (all ticked by default) with "Delete all" — untick any and it becomes "Delete selected", or choose "Don't delete" to keep them. Shared dependencies are never removed.
- **More languages**: the launcher is now available in Russian and China — auto-detected on first run.

## Changes (v0.3.0)

### ✨ Features & Improvements
- **More languages**: the launcher is now available in German, Spanish and French, alongside English and Polish — auto-detected on first run.
- **World management**: back up a world to a `.zip` or delete it, straight from the Worlds page and the instance's Worlds tab.
- **Screenshot deletion**: remove screenshots from the lightbox on both the Screenshots page and the instance tab.
- **Enable/disable resource packs & shaders**: toggle them on or off (just like mods) instead of only deleting.
- **Anonymous statistics (opt-in)**: an optional, privacy-friendly usage stats toggle in Settings → Privacy. Off by default, no personal data — it just helps prioritise which loaders and Minecraft versions to support.

## Changes (v0.2.0)

### ✨ Features & Improvements
- **Worlds hub**: a new Worlds page lists every world across all your instances, grouped by instance, with one-click Quick Play to jump straight into a world (MC 1.20+).
- **Screenshots gallery**: a new Screenshots page collects screenshots from all instances, grouped by instance, with a full-screen lightbox (arrow-key navigation, download, reveal in explorer).
- **Skins page redesign**: saved skins, the add button and the "in use" marker now sit together next to the default skins for a cleaner layout.
- **Skeleton loading everywhere**: smooth placeholder loaders while content loads — on the home library, the sidebar instance list, instance tabs (mods, resource packs, shaders, screenshots, worlds) and the skins/capes views.

## Changes (v0.1.2)

### ✨ Features & Improvements
- **New name**: Mako Launcher is now **Spectra Launcher** — fresh branding to match the spectrum-glow logo.
- **Update in the titlebar**: the title bar now shows a live download progress bar with a percentage while the launcher updates, plus a one-click "Update" button that appears the moment a new version is available.
- **Cleaner Settings**: the launcher info and update controls are pinned to the bottom; only the settings content scrolls, and the left sidebar stays in place.

### 🐛 Bug Fixes
- **Auto-update actually works now**: fixed a bug where clicking "Update" did nothing (the update object was being broken by Vue reactivity). Update failures now show a clear message instead of failing silently.

## Changes (v0.1.0)

### ✨ Features & Improvements
- **Auto-Update**: Discord-style background updater — click "Update" in Settings and the new version downloads, installs silently and relaunches with no installer clicks.
- **CurseForge Support**: Browse, install and update CurseForge mods, modpacks, resource packs and shaders. Blocked mods get a guided download flow.
- **Modrinth & CurseForge Export**: Export instances as `.mrpack` or CurseForge `.zip` with a Prism-style file tree. Partial selection lets you exclude a folder yet re-include a single child (e.g. only `config/fancymenu`).
- **Bulk Updates**: "Update all" now uses bulk API endpoints to avoid rate limits.
- **Instance Tweaks**: Change the mod loader after creation, change the instance icon by clicking it, and mods with available updates sort to the top.

### 🐛 Bug Fixes
- Fixed "last played" not updating — it's now saved at launch start.
